## 🧪 Paramètre requis

| Nom       | Type   | Description                     |
|----------|--------|---------------------------------|
| userName | string | Nom de l'utilisateur à tester   |

---

## 👤 Utilisateurs de test

En mode test, l'application génère automatiquement des utilisateurs internes.

👉 Format :

internalUserX

où **X est un entier** (0 → N)

---

## ✅ Exemples valides

- internalUser0  
- internalUser1  
- internalUser2  
- internalUser10
